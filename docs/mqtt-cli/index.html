DutchMeshCore Firmware — dutchmeshcore.nl

Dit hulpmiddel is speciaal voor de DutchMeshCore fork van MeshCore-MQTT. De standaard presets en instellingen zijn afgestemd op het Nederlandse meshnetwerk.

Bezoek dutchmeshcore.nl

MeshCore MQTT Instelling

Stel je MQTT-slots in en krijg kant-en-klare CLI-opdrachten voor je node.

Hoe gebruik je dit hulpmiddel▼

Stappen

1. Kies je IATA-luchthavencodes (bijv. AMS voor Schiphol) — dit geeft je locatie aan op de meshkaart.
2. Kies een preset voor elk slot dat je wil inschakelen. Slots op none zijn uitgeschakeld.
3. Voor MeshRank-slots, plak je token van meshrank.net.
4. Kopieer de gegenereerde opdrachten en plak ze in de MeshCore CLI (Remote Terminal of seriële console).

Standaard slots in DutchMeshCore firmware

De DutchMeshCore build stelt standaard slot 1 → dutchmeshcore-1 en slot 2 → dutchmeshcore-2 in. Je hoeft alleen set mqtt.iata <code> uit te voeren om te beginnen met publiceren — tenzij je de slottoewijzingen wil wijzigen.
